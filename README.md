# Northern Tier Energy (northern-tier-energy)

Northern Tier Energy was an independent downstream energy company with refining, retail, and pipeline operations. It was acquired by Western Refining in 2016, which itself merged into Marathon Petroleum's brand portfolio. No active public APIs are associated with this entity; the repository is maintained as a historical record within the API Evangelist Network.
